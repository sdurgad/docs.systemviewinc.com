AWS F1

Visual System Integrator on AWS EC2

Requirements

  • An active AWS account with billing and the ability to launch paid instance.
  • To use GUI, a local installation of NoMachine Client on the local computer. (Download here)

Launching Visual System Integrator on AWS

Launch a VSI FPGA Developer Instance using AWS console or cli tool.1

Once the the instance is running, launch the NoMachine client and configure it to connect to the public DNS name or IP of the instance.2

NoMachine client can be configured to use the same private key that you configured when you launched the instance3. Press connect and wait. It might take a minute or two to establish the connection. If you see a prompt asking if a new X-server should be launched, select yes.

  2. Getting started with NoMachine
